# Supplied IEEE/GLOBECOM LaTeX examples

These `.tex` and `.bib` files were extracted from `TexExamples.zip`, supplied by Guillem on 2026-07-14. They are retained in this private repository as drafting and formatting references. Large figures, generated files, nested ZIP archives, and duplicate `IEEEtran.cls` copies were intentionally omitted.

Primary source files:

- `spatially_consistent/main.tex`: recent Evgenii Vinogradov first-author conference structure; the strongest source-level style anchor.
- `Drone_ID/main.tex`: Evgenii Vinogradov GLOBECOM Workshops paper source.
- `3D_beamforming/main.tex`: six-page UAV/beamforming paper source with explicit background, state-of-the-art, problem, and contributions subsections.
- `Handover/FinalSubmission.tex`: compact result-led abstract and method/results organization.
- `Multi_Operator_Diversity/no_sl_main.tex`: UAV multi-operator evaluation paper source.

Use these files to learn document structure, equation density, caption conventions, and argument flow. Do not copy prose. The current live venue rules and the repository's installed `IEEEtran` package take precedence over old local class files.
