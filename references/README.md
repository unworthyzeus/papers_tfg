# Reference library

The `papers/` directory contains openly reachable study copies used to understand technical context, structure, and IEEE six-page density. The `venue/` directory contains official venue calls. The `source_examples/` directory contains the `.tex`/`.bib` examples supplied by Guillem; these are now the highest-weight references for source-level IEEE/GLOBECOM formatting.

`manifest.csv` records the source URL, page count, purpose, and SHA-256 checksum for every PDF. The library was validated as PDF content after download.

Priority when drafting:

1. Supplied Genia/GLOBECOM LaTeX sources, especially `spatially_consistent/main.tex` and `Drone_ID/main.tex`.
2. Recent Genia papers on spatially consistent A2G modeling.
3. GLOBECOM dense radio-map and channel-modeling examples.
4. Broader channel-modeling tutorials and white papers for terminology and literature orientation.

Copyright note: these files are retained in this private research repository for study. Before making the repository public or redistributing any PDF, verify the license of each version and replace restricted files with bibliographic links when necessary.
