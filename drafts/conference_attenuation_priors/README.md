# Conference attenuation-prior draft

Six-page-target IEEE conference/workshop manuscript derived from the attenuation-prior part of the thesis. It deliberately excludes HARP-Net, delay spread, and angular spread so that the journal extension remains materially distinct.

The `data/` directory contains the exact NLoS coefficient export and a
held-out term audit generated by `scripts/analyze_nlos_terms.py`. The audit
reports feature means and coefficient-times-feature contributions because raw
multilinear-regression coefficient magnitudes are not comparable across
differently scaled features.

Build with:

```powershell
latexmk -pdf -interaction=nonstopmode -halt-on-error paper.tex
```

The current numerical results are copied from the validated thesis evaluation. Before submission, add the ablation requested in `planning/paper_portfolio.md` and verify all numbers directly from exported artifacts.
