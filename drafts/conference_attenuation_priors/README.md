# Conference attenuation-prior draft

Six-page-target IEEE conference/workshop manuscript derived from the attenuation-prior part of the thesis. It deliberately excludes HARP-Net, delay spread, and angular spread. A companion paper will apply analogous interpretable prior techniques to both spread quantities, keeping the later multi-output contribution materially distinct.

The `data/official_split_analysis/` directory is generated by
`scripts/run_conference_attenuation_ablation.py`. It contains the official
Try 74/75 compatible split contract, refitted NLoS calibrations, component
ablation, per-map and per-city results, city bootstrap intervals, explicit
ground-only visibility shares, the configured-ray-tracer visibility contract,
exact coefficient and applied-term audits, and runtime metadata. The NLoS raw
term is COST231 only.

Regenerate all quantitative artifacts from the repository root with:

```powershell
& C:\TFG\.venv\Scripts\python.exe .\scripts\run_conference_attenuation_ablation.py
& C:\TFG\.venv\Scripts\python.exe .\scripts\analyze_nlos_terms.py
```

Build with:

```powershell
latexmk -pdf -interaction=nonstopmode -halt-on-error paper.tex
```

The paper tables are tied to the generated artifacts above. The HDF5 dataset
is read only and is not distributed with this paper repository.
